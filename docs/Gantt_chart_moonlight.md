## 8. Project Management Notes

The project follows a structured sequence from planning to final submission.

The schedule is organised so that:

- Planning is completed before research.
- Research supports requirements.
- Requirements support architecture.
- Architecture supports UI and system design.
- Design supports diagrams.
- Diagrams support database design.
- Database design supports security and testing.
- All sections are compiled into the final SRS before submission.
