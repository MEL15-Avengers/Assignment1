## 1. Purpose

This document explains the wireframes created for the Moonlight Warehouse Inventory Management System.

The wireframes show the planned user interface screens, page layouts, navigation structure, and main system functions.

## 6. Key Status Labels

The wireframes include the following inventory status labels:

| Status | Meaning |
|---|---|
| In Stock | Product is available |
| Low Stock | Product quantity is below threshold |
| Out of Stock | Product quantity is zero |
| Expiring Soon | Product batch is close to expiry |
| Expired | Product batch has passed expiry date |
| OK | Product batch is valid |

The wireframes provide a complete visual plan for the Moonlight Warehouse Inventory Management System.
