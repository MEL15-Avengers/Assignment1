## 1. Budget Objective

This budget template is for the Moonlight Warehouse Inventory Management System capstone project. It is designed to present a realistic and professional financial plan for the full project lifecycle.

The budget covers:

- Planning
- Design
- Development
- Testing
- Deployment
- Training
- Maintenance
- Contingency
