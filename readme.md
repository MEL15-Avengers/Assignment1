## Project Overview
The **Moonlight Warehouse Inventory Management System** is a web-based solution designed to improve warehouse operations by providing secure, centralized, and efficient inventory control. The system focuses on reducing manual stock handling errors, improving product visibility, supporting supplier and stock movement management, and enabling better operational decision-making.

This project is being developed as part of **Assignment 1**, with emphasis on planning, research, requirements analysis, and system design documentation.

---

## Purpose of the Project
The purpose of this project is to design and develop a warehouse inventory management system that helps warehouse staff and managers track products, monitor stock levels, manage suppliers, record stock movements, and improve overall inventory accuracy.

The system aims to replace inefficient manual or disconnected processes with a centralized digital platform.

---

## Project Objectives
- Improve inventory visibility and stock accuracy
- Support secure login and role-based access control
- Manage products, categories, and suppliers efficiently
- Record stock in, stock out, damaged stock, and inventory adjustments
- Provide alerts for low-stock and expiry-related risks
- Support reporting and operational monitoring
- Build a structured and professional system design for implementation

---

## Scope
The project includes:
- User authentication and authorization
- Product management
- Category management
- Supplier management
- Inventory management
- Stock movement tracking
- Batch and expiry tracking
- Alerts and dashboard summaries
- Database design and system documentation

The project excludes:
- Full ERP integration
- Advanced forecasting
- Online customer ordering
- Delivery routing and logistics optimization
- Enterprise-scale external integrations
